Function for replacing the background in images using the WorqHat API. It sends a request to the Replace Background in Image AI Model and returns the modified image. Read more about the model at https://docs.worqhat.com/ai-models/image-generation/image-image-v3

### Parameters

| Parameter    | Type               | Description                                                                                                             |
| ------------ | ------------------ | ----------------------------------------------------------------------------------------------------------------------- |
| image        | `File` or `string` | The image to which the background will be replaced. It can be a `File object`, a `URL`, or `base64` encoded image data. |
| modification | `string`           | The description of the modification for replacing the background. This is a required parameter.                         |
| output_type  | `string`           | The output type of the modified image. It can be either `"URL"` or `"blob"`.                                            |
| api_key      | `string`           | Contains the WorqHat API Key. Not required if .env is made.                                                             |

### Initialize the library

```bash .env
API_KEY="Your API Key"
```

You can also provide the `api_key` as a parameter to the function as shown in the implementation.
