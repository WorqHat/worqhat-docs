Function for modifying images using version 3 of the API. It sends a request to the Image Modification V3 AI Model and returns the new image. Read more about the Model at https://docs.worqhat.com/ai-models/image-generation/image-image-v3

### Parameters

| Parameter      | Type               | Description                                                                                                                          |
| -------------- | ------------------ | ------------------------------------------------------------------------------------------------------------------------------------ | ----------- |
| existing_image | `File` or `string` | The existing image to be modified. It can be a `File object`, a `URL`, or `base64` encoded image data. This is a required parameter. |
| modification   | `string`           | The modifications to be made to the image. This is a required parameter.                                                             |
| outputType     | `url` `blob`       | The output type of the modified image. It can be either `url` or `blob`. This is an optional parameter.                              |
| similarity     | `number`           | The similarity percentage for the image modification. This is a required parameter.                                                  |
| api_key        | `string`           | Contains the WorqHat API Key. Not required if .env is made.                                                                          | `undefined` |

### Initialize the library

```bash .env
API_KEY="Your API Key"
```

you can also provide the api_key as a parameter to the function as shown in the implementation

### Implementation

```python
from worqhat import modify_image_v3
from dotenv import load_dotenv
load_dotenv()
image="path-to-image/URL-to-image"
r=modify_image_v3(image=image,modification="changes",api_key="Your-api-key")
print(r)

```
