Function for training a dataset using the WorqHat API. It sends a request to the Train Dataset endpoint and initiates the training process for the specified dataset. Read more about the API at the WorqHat documentation.

### Parameters

| Parameter     | Type     | Description                                                 |
| ------------- | -------- | ----------------------------------------------------------- |
| dataset_id    | `string` | The ID of the dataset to be trained.                        |
| dataset_name  | `string` | The name of the dataset.                                    |
| dataset_type  | `string` | The type of the dataset.                                    |
| json_data     | `string` | JSON data related to the dataset.                           |
| training_file | `File`   | The training file to be used for training the dataset.      |
| api_key       | `string` | Contains the WorqHat API Key. Not required if .env is made. |

### Initialize the library

```bash .env
API_KEY="Your API Key"
```

You can also provide the `api_key` as a parameter to the function as shown in the implementation.

### Implementation

```python
import os
import requests
from dotenv import load_dotenv

# Load API Key from .env file
load_dotenv()

# Set the dataset ID, name, type, and JSON data
dataset_id = "your-dataset-id"
dataset_name = "Your Dataset Name"
dataset_type = "Your Dataset Type"
json_data = "Your JSON Data"

# Set the path to your training file
training_file_path = "path-to-your-training-file"

# Get API key from environment variables
api_key = os.getenv("API_KEY")

# Train dataset
result = train_dataset(dataset_id=dataset_id, dataset_name=dataset_name, dataset_type=dataset_type, json_data=json_data, training_file=training_file_path, api_key=api_key)

# Print the result (success message or error)
print(result)
```

Remember to replace `"Your API Key"`, `"your-dataset-id"`, `"Your Dataset Name"`, `"Your Dataset Type"`, `"Your JSON Data"`, and `"path-to-your-training-file"` with your actual API key, dataset ID, dataset name, dataset type, JSON data, and the path to your training file, respectively.
