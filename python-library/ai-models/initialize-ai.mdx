---
title: "Initialize AI Models"
description: "Initialize AI Models to use them in your app"
---

Initialize AI models using the `worqhat` library in your application. It demonstrates how to create a configuration object with an API key and debug mode, initialize the `worqhat` app with the configuration, and access the AI functionality through the worqhat library.

You need to install the `worqhat` library to run the code in this file and import the `worqhat` library.

### Example

```python
from worqhat.ai_models import text_gen
```
