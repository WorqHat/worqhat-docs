**Large Content Generation AI Model.** This model is designed to generate large amounts of content based on the provided parameters. You can provide a pre-trained model to generate content on top of it. This is mostly used for use-cases where you want to run a model based on your pre-trained dataset. It can be used to generate content for a variety of use cases where the content is more creative or complex.

Read more at https://docs.worqhat.com/ai-models/text-generation-ai/aicon-v2-large-beta

### Parameters

| Parameter            | Type     | Description                                                      |
| -------------------- | -------- | ---------------------------------------------------------------- |
| question             | `string` | The question or input for which AI response is requested.        |
| dataset_id           | `string` | The ID of the dataset to be used for generating the AI response. |
| preserve_history     | `bool`   | Whether to preserve conversation history for context.            |
| randomness           | `float`  | The level of randomness for generating AI response.              |
| stream_data          | `bool`   | Whether to stream response data.                                 |
| conversation_history | `list`   | List containing conversation history for context preservation.   |
| instructions         | `string` | Additional instructions for the AI model.                        |
| api_key              | `string` | Contains the WorqHat API Key. Not required if .env is made.      |

### Initialize the library

```bash .env
API_KEY="Your API Key"
```

You can also provide the api_key as a parameter to the function as shown in the implementation.
###Implementation

```python
import os
import requests
import json
from dotenv import load_dotenv

# Load API Key from .env file
load_dotenv()

# Set the parameters for the AI request
question = "Your question or input here"
dataset_id = "Your dataset ID here"
preserve_history = True
randomness = 0.5
stream_data = False
conversation_history = []  # If preserving conversation history
instructions = None  # Additional instructions for the AI model

# Get API key from environment variables
api_key = os.getenv("API_KEY")

# Get large AI response
get_large_ai_response_v2(question=question,
                         dataset_id=dataset_id,
                         preserve_history=preserve_history,
                         randomness=randomness,
                         stream_data=stream_data,
                         conversation_history=conversation_history,
                         instructions=instructions,
                         api_key=api_key)
```
