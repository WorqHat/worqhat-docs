---
title: "AiCon V3 Advanced"
description: "Advanced Generation AI focused for more creative and understanding capabilities."
---

The AiCon V3 Advanced is a generation AI model that is designed to be more creative and have a better understanding of complex content. It is suitable for a wide range of use cases where the content requires creativity and complexity. This model is capable of running complex situational analysis and understanding the context of commands.

Read more at https://docs.worqhat.com/ai-models/text-generation-ai/aicon-v3-textgen

### Parameters

| Parameter        | Type      | Description                                                                  | Default     |
| ---------------- | --------- | ---------------------------------------------------------------------------- | ----------- |
| history_object   | `Object`  | An object representing the conversation history.                             | `undefined` |
| preserve_history | `boolean` | A boolean indicating whether to preserve the conversation history.           | `false`     |
| question         | `string`  | A string representing the question to generate content for.                  | `undefined` |
| training_data    | `string`  | A string representing the training data to use for generating content.       | `undefined` |
| randomness       | `number`  | A number representing the level of randomness for generating content.         | `0.2`       |


### Example

Here is an example of how to use AiCon V3 Advanced to generate content:

```javascript
const worqhat = require("worqhat");

var config = new worqhat.Configuration({
  apiKey: "your-api-key",
  debug: true,
});

worqhat.initializeApp(config);

async function generateContent() {
  try {
    var result = await worqhat.contentGeneration.v3({
      history_object: { previous: "Previous conversation history" },
      preserve_history: true,
      question: "Your question here",
      training_data: "your-training-data",
      randomness: 0.3,
    });
    console.log(result);
  } catch (error) {
    console.error(error);
  }
}

generateContent();
```
