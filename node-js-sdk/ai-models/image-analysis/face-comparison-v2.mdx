---
title: "Comparing Faces with AI"
description: "Detect faces in images and compare them with other faces in a Database or another image."
---

The Face Comparison AI Model by WorqHat, powered by ImageCon V2, is designed to determine if a face in one image matches a face in another image. This system compares faces regardless of variations in expression, facial hair, and age. It provides confidence level estimates for predictions using probabilities or confidence scores. Users should consider the confidence threshold appropriate for their specific use case to balance the trade-off between missed detections and false alarms. The Face Comparison AI Model by WorqHat, powered by ImageCon V2, offers the tools and capabilities to compare faces and determine potential matches.

Read more at https://docs.worqhat.com/ai-models/image-analysis/face-analysis

### Parameters

| Parameter    | Type               | Description                                                                                                                           |
| ------------ | ------------------ | ------------------------------------------------------------------------------------------------------------------------------------- |
| source_image | `File` or `string` | The existing image to be upscaled. It can be a `File object` or a `URL` or `base64` encoded image data. This is a required parameter. |
| target_image | `File` or `string` | The existing image to be upscaled. It can be a `File object` or a `URL` or `base64` encoded image data. This is a required parameter. |

### Examples

```js
const worqhat = require("worqhat");

var config = new worqhat.Configuration({
  apiKey: "your-api-key",
  debug: true,
});

worqhat.initializeApp(config);

async function compareFaces() {
  try {
    var result = await worqhat.analyseImages.compareFaces({
      source_image: "./path-to-source-image.png",
      target_image: "./path-to-target-image.png",
    });
    console.log(result);
  } catch (error) {
    console.error(error);
  }
}

compareFaces();
```
